# ssunny_everytime
이전에 적용해본 기능들을 사용한 로그인/게시판 기능 및 주 목적인 시간표를 설정하고 조회할 수 있는 사이트를 제작하고자 함.
+React Native를 이용하여 추가적으로 앱 개발 또한 진행(2024.3.2)
##
# 배포시 유의사항
- Front
  - env파일 백업
- Back
  - env파일 백업
- App
  - 개발 후 추후 추가예정
